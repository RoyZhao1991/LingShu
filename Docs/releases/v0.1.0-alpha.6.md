# LingShu v0.1.0-alpha.6

[English](#english) | [简体中文](#简体中文)

## English

This alpha turns LingShu's artifact workflow into something prospective users can inspect before installing. It adds a privacy-safe, reproducible Project Aurora sample and closes two visible English-interface gaps in command approval and task participant categories.

### What changed

- Added an editable four-slide Project Aurora PPTX, a one-page DOCX, a PDF preview, and rendered slide images built entirely from fictional data.
- Added direct sample links and previews to the bilingual README and project website.
- Fixed light-mode command approval text so unselected actions remain readable.
- Localized system-owned task participant and role labels at display time without changing persisted task history, filtering, or lineage.
- Added regression coverage for the English participant categories and built-in role titles.

### Verification and release integrity

- Full local suite: 1,581 tests executed, 4 environment-dependent tests skipped, 0 failures.
- GitHub CI passed on release source commit `af829f1eeac627cde362f23e6b799a0101b2f521`.
- The public PPTX has exactly four 16:9 slides; the DOCX renders as one page; both OOXML archives passed integrity and privacy scans.
- Build 9 was compiled from a clean source tree for both Apple silicon and Intel.
- The app and DMG were separately accepted by Apple notarization and carry stapled tickets.
- The release script remounted the DMG and revalidated the installed app, bundled CLI, signature, notarization, and embedded source revision.
- DMG SHA-256: `1e3ad6b3311e69c6e83e2da69b3b049eeb400ace81e0d8b82dc99812fd2d7d81`.

### Install

```bash
brew install --cask RoyZhao1991/tap/lingshu
```

Or download the Universal DMG and verify it:

```bash
shasum -a 256 -c LingShu-0.1.0-9-macOS-universal.dmg.sha256
```

Requires macOS 14 or later and a supported model token or compatible custom endpoint. LingShu remains alpha software; keep backups and review high-impact actions and generated deliverables.
