# LingShu v0.1.0-alpha.8

[English](#english) | [简体中文](#简体中文)

## English

This alpha adds a reproducible clean-user installation gate for LingShu's public macOS package. The check starts from the same Apple-notarized DMG that users download, installs it into a disposable environment, and verifies first launch without reading the maintainer's saved settings, credentials, or task history.

### What changed

- Adds an opt-in clean-user runtime boundary with isolated home, Application Support, preferences, temporary files, and in-memory credentials.
- Disables Keychain access, permission services, background services, and environment-token fallback inside that disposable smoke mode.
- Verifies the real language-selection and no-brain setup views appear on first launch.
- Runs a deterministic local direct reply without a configured model, while confirming the app stays alive.
- Adds focused tests for the isolation contract and rejects unsafe or mismatched smoke roots.
- Publishes the machine-readable clean-user result alongside the release package.

### Verification and release integrity

- GitHub CI passed on release source commit `081976ba8f7d64d803332d0b3b0cd889998545ad`.
- Full local suite: 1,586 tests executed, 4 environment-dependent tests skipped, 0 failures.
- Focused clean-user suite: 4 tests, 0 failures.
- Build 11 was compiled from a clean source tree for both Apple silicon and Intel.
- The app and DMG were separately accepted by Apple notarization and carry stapled tickets.
- The notarized DMG passed Gatekeeper, checksum verification, disposable installation, signature validation, first-run UI checks, no-permission checks, empty-history checks, and the minimal direct-reply check.
- All 11 machine-readable clean-user checks are `true`; inspect `clean-user-smoke-result.json` in the release assets.
- DMG SHA-256: `815a1e8471e1dfc2cd2e773f388567601fc36fb45d30b83eec4c4743644822d4`.

### Install

```bash
brew install --cask RoyZhao1991/tap/lingshu
```

Or download the Universal DMG and verify it:

```bash
shasum -a 256 -c LingShu-0.1.0-11-macOS-universal.dmg.sha256
```

Requires macOS 14 or later and a supported model token or compatible custom endpoint for model-backed tasks. LingShu remains alpha software; keep backups and review high-impact actions and generated deliverables.

## 简体中文

本次 Alpha 为灵枢公开 macOS 安装包补上可复现的“全新用户安装”硬门。验证从用户实际下载的 Apple 公证 DMG 出发，在一次性隔离环境中安装并启动，且不会读取维护者已有的配置、凭据或任务历史。

### 本次改动

- 增加显式启用的全新用户隔离运行边界，隔离用户目录、Application Support、偏好设置、临时文件与内存凭据。
- 在该一次性验证模式中关闭钥匙串读取、权限服务、后台服务和环境变量 Token 回退。
- 验证首次启动真实出现语言选择页和无主脑配置引导页。
- 在未配置模型时执行确定性的本地直答，并确认 App 持续存活。
- 增加隔离边界专项测试，并拒绝不安全或不一致的测试目录。
- 将机器可读的全新用户验证结果与安装包一同发布。

### 验证与发布完整性

- 发布源码提交 `081976ba8f7d64d803332d0b3b0cd889998545ad` 的 GitHub CI 已通过。
- 本地完整测试：执行 1,586 项，4 项环境相关测试按设计跳过，0 失败。
- 全新用户专项测试：4 项，0 失败。
- Build 11 从干净工作区构建，同时覆盖 Apple 芯片与 Intel。
- App 与 DMG 分别通过 Apple 公证并装订公证票据。
- 公证 DMG 依次通过 Gatekeeper、校验和、一次性安装、签名、首次启动界面、无权限基线、空任务历史和最小直答验证。
- 机器可读结果中的 11 项检查全部为 `true`，可在 Release 资产 `clean-user-smoke-result.json` 中查看。
- DMG SHA-256：`815a1e8471e1dfc2cd2e773f388567601fc36fb45d30b83eec4c4743644822d4`。
